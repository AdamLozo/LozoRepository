# DAX Library

A collection of optimized DAX measures and patterns following SQLBI methodology.

## Categories

### Time Intelligence
DAX measures for date-based calculations and comparisons.

### Financial Metrics
Measures for ARR, bookings, cash flow analysis.

### HR Analytics
Measures for headcount, turnover, and tenure analysis.

### Aggregations
Common aggregation patterns and calculations.

### Utilities
Helper measures and calculation patterns.

## Usage

Each category contains documented measures with:
- Measure name and description
- DAX code
- Usage examples
- Performance considerations
- Dependencies

## Naming Convention

Follows SQLBI standards:
- Descriptive names with spaces
- Prefixes for measure types where appropriate
- Clear indication of calculation context
